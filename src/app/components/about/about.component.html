About the project Form builder by Amit Turner

In order to build this project I had to learn Angular, node.js and mongo from scrach. I chose to use this stack after reading online and deciding I will have the best result considering there are 2 weeks to learn the platform and to make the product. Angular and node are well known and there are many examples to learn from, as well as many solutions for various errors and mistakes I encounter as a first project in this stack.

I tried to give the user a nice intuative UX while making sure the server actions are clean, fast and as simple as possible. this while maintaining all the database in a single object array and not dividing it to "form data" and "submissions data" even if it might be a bit faster to query.
